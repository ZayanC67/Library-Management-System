# Library-Management-System
A web-based Library Management System developed using JSP, Servlets, MySQL and Apache Tomcat.


## Project Overview

This project is developed using:

- JSP
- Servlets
- MySQL
- Apache Tomcat

## Features

- User Login
- Book Management
- Issue Book
- Return Book
- Search Books
